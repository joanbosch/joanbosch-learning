# Tipus de dades

<Callout emoji="">
  En Python, cada valor que es fa servir és d'un tipus concret. Per exemple, el valor 38 és de tipus enter (*int*), el valor 3.14 és de tipus real (*float*) i el valor 'python' és de tipus text (*str*).
  Per saber el tipus d'un valor podem fer servir la funció `type()`.
</Callout>

```python
>>> type(38)
<class 'int'>
>>> type(3.14)
<class 'float'>
>>> type('python')
<class 'str'>
```

## Els tipus bàsics de Python

Amb Python, tenim quatre tipus de dades bàsics:

1. **El tipus enter (`int`)**: Usat per representar números enters.
2. **El tipus real (`float`)**: Usat per representar números decimals.
3. **El tipus text (`str`)**: Usat per representar cadenes de text.
4. **El tipus booleà (`bool`)**: Els valors del tipus de dades booleà poden ser `true` o `false`.

Evidentment, existeixen altre tipus simples i complexes, però en aquest curs en centrarem amb l'ús d'aquests tipus més senzills per poder-nos introduïr al món de la programació.
