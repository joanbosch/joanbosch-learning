# Operadors

<Callout emoji="">

## Operadors aritmétics
Els operadors aritmétics són aquells que ens ajuden a executar operacions aritmétiques. En conseqüència, un operador aritmètic agafa dos operands d'entrada, realitza un càlcul i retorna el seu resultat aritmètic.

Per exemple, considerem l'expressió `suma = 2 + 3` que hem vist en aparatas anteriors. En aquest cas,`2` i `3` són els operands i `+` és l'operador aritmètic. El resultat de l'operació es guarda a la variable `suma`.

A continuació podem veure els operadors aritmètics més comuns:

| Operador        | Descripció |Ús |
| :------------: | :---------: | :----------: |
|+|Realitza una suma entre els operands| `2 + 2 # 4` |
|-|Realitza una resta entre els operands| `2 - 2 # 0` |
|*|Realitza una multiplicació entre els operands        | `2 * 2 # 4` |
|/|Realitza una divisió entre els operands | `2 / 2 # 1` |
|%|Realitza un módul entre els operands| `4 % 2 # 0` |
|**|Realitza la poténcia dels operands| `2 ** 3 # 8` |
|//|realitza una divisió dels operands amb un resultat enter| `3 // 2 # 1` |
